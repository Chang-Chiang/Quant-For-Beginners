## 1. 股票相关基本概念

- 量化金融：用数学、统计学、计算机科学和金融理论，通过建立模型和算法分析数据来进行交易和风险管理
- 核心逻辑：不预测未来，只统计过去、验证规律、寻找概率优势
- 量化四步闭环：**提出假设 → 量化定义 → 历史验证 → 沉淀规则迭代**
- 完整流程：获取数据 → 分析数据 → 设计策略 → 回测策略 → 模拟交易
- Python 是量化首选语言：易学、生态丰富、AI 支持、免费

**历史里程碑**：
- 1900 巴舍利耶：股价随机游走与布朗运动（理论原点）
- 1961 索普：21 点算牌实验（第一个可复制可盈利的量化实验）
- 索普公式：概率优势 × 仓位管理 × 长期纪律 = 量化盈利

**OHLCV 数据**：Open / High / Low / Close / Volume，最常用 Close（收盘价）。

**收益率**：

$$
r_t = \frac{P_t - P_{t-1}}{P_{t-1}}
$$

- 收益率 = 价格变化占昨天价格的比例
- `df['ret'] = df['Close'].pct_change()`
- 累计收益率：`(1 + rets).cumprod() - 1`

**风险指标**：
- **波动率** ≈ 收益率的标准差 σ，年化：`daily_vol * np.sqrt(252)`
- **夏普比率** = (年化收益 − 无风险利率) / 年化波动率 → 衡量「性价比」
- **Beta** = Cov(股票, 大盘) / Var(大盘) → 衡量相对大盘的敏感度
- **最大回撤** = 从历史高点到最深谷底的跌幅 → 衡量「最惨时你会经历什么」

**组合管理**：
- 相关系数 ρ：量化「同涨同跌」，ρ ≈ 0 是分散化理想素材
- 等权组合：成分不完全相关时，组合波动往往低于单票（分散化红利）
- 仓位管理：策略决定方向，仓位决定生死。先定风险预算，再反推仓位

## 3. 数据分析

**移动平均线（MA）**：

$$
MA_n(t) = \frac{1}{n} \sum_{i=0}^{n-1} P_{t-i}
$$

- `df['MA20'] = df['Close'].rolling(20).mean()`
- n 越小更敏感（跟得紧但假信号多），n 越大更平滑（迟钝但稳）

**金叉 / 死叉**：
- 金叉：MA5 从下往上穿过 MA20 → 偏多
- 死叉：MA5 从上往下穿过 MA20 → 偏空
- 检测：`np.sign(df['MA5'] - df['MA20']).diff()`

**波动率计算**：

$$
\sigma = \sqrt{\frac{1}{n-1}\sum_{i=1}^{n}(r_i - \bar{r})^2}
$$

- 玩具验证：B 的标准差远大于 A → 和肉眼看到的「过山车」一致
- 三票 PK：TSLA 通常最高（最刺激），AAPL 通常最低（最稳）

**相关性矩阵**：
- `corr = returns.corr()` → Pearson 相关系数 ρ
- 热力图：`imshow(cmap='RdBu_r')`，深红 = 强正相关，深蓝 = 负相关
- 科技双子星（AAPL-MSFT）通常 ρ 0.75+，XLE 和科技股更「冷」

**可视化三件套**：

1. **收盘价 + 成交量**（上下子图）
   - 上图看价格走势：整体向上 = 上涨趋势，整体向下 = 下跌趋势
   - 下图看成交量：放量（柱子突然变高）常伴随重要事件或趋势转折
   - 成交量萎缩 = 市场分歧减小，趋势可能延续

2. **日收益率曲线 + 直方图（Histogram）**
   - 曲线图：每天涨跌幅度一目了然，极端值（远离零轴）代表异常波动日
   - 直方图：大部分日子涨跌集中在哪？又窄又高 = 平稳，又宽又胖 = 波动大
   - 直方图尾巴很长 = 厚尾分布，意味着极端涨跌比正态分布预期的更频繁（Mandelbrot 1963 发现）

3. **累计收益率曲线**
   - 回答「这一段总体赚还是亏？」
   - `(1 + rets).cumprod() - 1`，起点 = 0，向上 = 赚钱，向下 = 亏钱
   - 曲线越平滑 = 持有体验越好；锯齿越多 = 心理压力越大

**进阶可视化**：

4. **策略净值曲线**（回测专用）
   - 三条线对比：策略 vs 买入持有（同标的）vs 大盘（SPY）
   - 策略在最上面 = 跑赢基准；在下面 = 跑输
   - 关注「坑」的深度和恢复速度——联系最大回撤

5. **水下曲线（Underwater Plot）**
   - 逐日回撤序列的填充图，净值在历史高点之下时填充越深
   - 关注深度（坑有多深）和宽度（在水下待多久才爬回水面）
   - 好的策略不仅要回撤浅，还要恢复快

6. **风险收益散点图**
   - 横轴 = 年化波动率（风险），纵轴 = 年化收益率（回报）
   - 左上角 = 低波动高收益（理想，少见），右下角 = 高波动低收益（性价比差）
   - 一眼看出各标的在风险收益谱系中的位置

7. **相关性热力图**
   - `imshow(cmap='RdBu_r')`：深红 = 强正相关（同涨同跌），深蓝 = 负相关，浅色 = 各走各的
   - 对角线永远 1.00（自己和自己），关注非对角线色块
   - 找最红的一格（最「双胞胎」）和最白/最蓝的一格（分散化候选）

## 4. 策略

**双均线策略（Moving Average Crossover）**：

```python
signal = (MA5 > MA20).astype(int)  # 1=持仓, 0=空仓
```

- 金叉买入、死叉卖出的最简版本
- 金叉不一定赚钱！均线是滞后指标，震荡市会频繁交叉产生假信号
- 短均线灵敏但信号多，长均线迟钝但信号少——没有「最好的」参数

**回测**：

> **回测是体检，不是算命。** 过去好 ≠ 未来一定好。

- 模拟交易：`position = signal.shift(1)`（信号推迟一天，避免未来函数）
- 策略收益：`strategy_ret = position * ret`
- 累计净值：`nav = (1 + strategy_ret).cumprod()`
- 三条净值曲线对比：策略 vs 买入持有（同标的） vs 买入持有（SPY 大盘）

**胜率**：完整「买入→卖出」回合中，赚的次数占比。胜率低不代表策略差——赢的时候赚得多也行。

**最大回撤**：

```python
peak = nav.cummax()
drawdown = nav / peak - 1
max_dd = drawdown.min()
```

**仓位管理**：
- 同一套 MA 规则，满仓 vs 半仓走的是两条不同的人生
- 满仓：收益和回撤都「满格」；半仓：大约都减半
- 先设定可承受回撤（如 −15%），再反推合理仓位

**等权组合**：
- 组合日收益 = 各成分日收益逐行平均：`returns[['A','B','C','D']].mean(axis=1)`
- 成分不完全相关时，组合波动低于单票（分散化红利）
- 用 SPY 基准对比：组合是在创造差异，还是重复大盘？

## 5. 根据策略买入卖出

**信号生成**：

```python
df['signal'] = (df['MA5'] > df['MA20']).astype(int)
df['position'] = df['signal'].shift(1).fillna(0).astype(int)  # 推迟一天
```

**标记买卖日**：

```python
df['position_change'] = df['position'].diff().fillna(0)
df.loc[df['position_change'] > 0, 'action'] = '买入'
df.loc[df['position_change'] < 0, 'action'] = '卖出'
```

**策略收益计算**：

```python
df['ret'] = df['Close'].pct_change().fillna(0)
df['strategy_ret'] = df['position'] * df['ret']
df['nav'] = (1 + df['strategy_ret']).cumprod()
```

**水下曲线（Underwater Plot）**：逐日回撤序列的填充图，同时看深度与在水下的时间。

**波动率伤人机制**：高波动 → 深回撤 → 恐慌割肉 → 拿不到理论收益。波动率不仅是数学指标，也是投资体验。

**组合 vs 单票**：

```python
port_ret = returns[['AAPL', 'MSFT', 'JPM', 'XLE']].mean(axis=1)
ann_vol = port_ret.std() * np.sqrt(252)
```
