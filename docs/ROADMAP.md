# 项目路线图（Roadmap）

> 定位：**中文最适合零基础的量化金融学习路线**  
> 原则：不追求讲完所有知识，只追求让更多人真正入门。

## 愿景

当一个从来没学过量化的人打开本仓库时，他应该在 30 秒内产生三个想法：

1. **我看得懂**
2. **我学得会**
3. **我先 Star 收藏**

## 传播与内容标准（每章检查清单）

- [ ] 开头：本章你将学会 + 等级 + 难度 + 时间 + 前置

- [ ] 结尾：🎯 挑战任务 + 本章总结
